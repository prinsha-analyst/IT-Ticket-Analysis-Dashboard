# IT Ticket Analysis Dashboard (Power BI)

## Project Overview

This project analyzes IT help desk ticket data to identify operational trends, ticket priorities, department workload, and service efficiency.

The dashboard provides insights that help IT teams improve response time and resource allocation.

---

## Dashboard KPIs

* **Total Users:** 967
* **Total Tickets:** 100,000
* **Total Departments:** 15


---

## Key Insights

### Ticket Severity

Most IT support tickets fall under **Normal severity**, indicating routine support operations.

### Priority Analysis

High priority tickets represent a significant portion of support workload.

### Department Workload

Departments such as **Tech Support, Finance, and HR** generate higher ticket volumes.

### Ticket Type

Most tickets are **Requests rather than Issues**, showing operational support demand.
